Code source du projet Ocean.s-26 Rajalga.

Vous trouverez :

- Dans le répertoire `arduino` les fichiers et les instructions pour utiliser les boutons et le joystick.
- Dans le répertoire `gdevelop` les fichiers et les instructions pour modifier et exporter le jeu-vidéo (en cas de modifications).
- Dans le répertoire `web` les fichiers et les instructions pour lancer le jeu-vidéo.

Les instructions se trouvent dans les fichiers `README.md`, directement lisibles depuis GitHub.

Pour télécharger tout le projet, cliquez sur `Code` puis `Télécharger le ZIP`. Ce fichier sera a décompresser pour utilisation.
